# 数字予測マジック

## 実行方法
```
$ sudo apt install python3-pyqt5
$ python3 magic100.py
```

## 遊び方
- magic100.py
  
  1-100の中で好きな数字を選んで下さい．  
  画面に表示される数字群の中に自分の選んだ数字があれば｢ある｣ボタンを，なければ｢ない｣ボタンをクリックして下さい．  
  何回か繰り返すと，あら不思議，自分の選んだ数字を見事に言い当てられる??

- magic1000.py
  
  1-1000の中で好きな数字を選んで下さい．
  あとはmagic100.pyの遊び方と一緒です．
